# Installation Guide\n\n## System Requirements\n\n### Development Environment\n- **Python**: 3.8 or higher\n- **Operating System**: Windows 10/11, macOS, or Linux\n- **Memory**: Minimum 8GB RAM (16GB recommended)\n- **Storage**: At least 2GB free space\n- **Internet**: Required for initial package installation\n\n### Mobile Development (Optional)\n- **Android Studio**: Latest version with Android SDK\n- **Android Device**: API Level 21+ (Android 5.0+) for testing\n- **USB Debugging**: Enabled on test device\n\n## Installation Steps\n\n### 1. Clone the Repository\n```bash\ngit clone https://github.com/[your-username]/reel-detector-project\ncd reel-detector-project\n```\n\n### 2. Python Environment Setup\n\n#### Using Virtual Environment (Recommended)\n```bash\n# Create virtual environment\npython -m venv .venv\n\n# Activate virtual environment\n# On Windows:\n.venv\\Scripts\\activate\n# On macOS/Linux:\nsource .venv/bin/activate\n\n# Upgrade pip\npip install --upgrade pip\n\n# Install dependencies\npip install -r requirements.txt\n```\n\n#### Using Conda (Alternative)\n```bash\n# Create conda environment\nconda create -n reel-detector python=3.8\nconda activate reel-detector\n\n# Install packages\npip install -r requirements.txt\n```\n\n### 3. Verify Installation\n```bash\n# Test Python packages\npython -c \"import tensorflow as tf; print('TensorFlow:', tf.__version__)\"\npython -c \"import pandas as pd; print('Pandas:', pd.__version__)\"\npython -c \"import sklearn; print('Scikit-learn:', sklearn.__version__)\"\n```\n\n### 4. Generate Synthetic Data\n```bash\n# Create necessary directories\nmkdir -p data models\n\n# Generate synthetic training data\npython notebooks/demo_synthetic_data.py\n```\n\n### 5. Train Initial Model\n```bash\n# Train and convert model to TFLite\npython notebooks/train_and_convert.py data/windows_labeled_synthetic.csv\n\n# Verify model creation\nls models/\n# Should show: model.h5, model_quant.tflite, scaler.json\n```\n\n### 6. Test Model Performance\n```bash\n# Evaluate TFLite model\npython notebooks/eval_tflite.py\n\n# Run simple tests\npython test_small.py\n```\n\n## Android Development Setup (Optional)\n\n### 1. Install Android Studio\n1. Download Android Studio from [developer.android.com](https://developer.android.com/studio)\n2. Install with default settings\n3. Open Android Studio and complete initial setup\n4. Install Android SDK (API Level 21 minimum)\n\n### 2. Import Android Project\n```bash\n# Navigate to Android project\ncd android\n\n# Copy trained models to assets\ncp ../models/model_quant.tflite app/src/main/assets/\ncp ../models/scaler.json app/src/main/assets/\n```\n\n### 3. Build Android App\n```bash\n# Build debug APK\n./gradlew assembleDebug\n\n# Install on connected device\n./gradlew installDebug\n```\n\n## Configuration\n\n### Environment Variables (Optional)\nCreate a `.env` file in the project root:\n```bash\n# Model configuration\nMODEL_PATH=models/model_quant.tflite\nSCALER_PATH=models/scaler.json\n\n# Data paths\nDATA_DIR=data\nOUTPUT_DIR=results\n\n# Training parameters\nEPOCHS=25\nBATCH_SIZE=64\nWINDOW_SIZE=5.0\n```\n\n### Custom Configuration\nEdit configuration files in `configs/` directory:\n- `model_config.json`: Model hyperparameters\n- `data_config.json`: Data processing settings\n- `android_config.xml`: Android app settings\n\n## Troubleshooting\n\n### Common Issues\n\n#### Python Package Conflicts\n```bash\n# Clear pip cache\npip cache purge\n\n# Reinstall packages\npip uninstall -y -r requirements.txt\npip install -r requirements.txt\n```\n\n#### TensorFlow Installation Issues\n```bash\n# For Apple Silicon Macs\npip install tensorflow-macos tensorflow-metal\n\n# For CUDA-enabled systems\npip install tensorflow-gpu\n\n# CPU-only fallback\npip install tensorflow-cpu\n```\n\n#### Android Build Errors\n```bash\n# Clean build\n./gradlew clean\n\n# Rebuild project\n./gradlew build --refresh-dependencies\n```\n\n#### Memory Issues\n- Increase JVM heap size: `export JAVA_OPTS=\"-Xmx4g\"`\n- Use smaller batch sizes in training scripts\n- Close other applications during training\n\n### Data Issues\n\n#### Missing Data Files\n```bash\n# Regenerate synthetic data\npython notebooks/demo_synthetic_data.py\n\n# Verify data creation\nls data/\n```\n\n#### Model Training Failures\n```bash\n# Check data format\npython -c \"import pandas as pd; df = pd.read_csv('data/windows_labeled_synthetic.csv'); print(df.shape, df.columns.tolist())\"\n\n# Retrain with verbose output\npython notebooks/train_and_convert.py data/windows_labeled_synthetic.csv --verbose\n```\n\n## Performance Optimization\n\n### For
